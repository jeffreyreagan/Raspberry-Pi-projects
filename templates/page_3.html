Technologies Used

PLC data collection

PyLogix, a Python library, enables communication with Allen-Bradley PLCs using the CIP protocol (Common Industrial Protocol). This library simplifies data exchange between the Python backend and PLCs, facilitating seamless integration and control.

Web Server

This web server is a simple Raspberry Pi 4b utilizing python's Flask library. Data storage is done on the server side with no SQL databases being used quite yet.

Flask servers can be set up locally or a full production application can be developed for remote connection applications.

Design

These SCADA systems are designed from scratch using html with svg drawings, allowing nearly unlimitted customization.

Visualization

SCADA systems at their core specialize in displaying critical information. Visualization is done using vanilla JavaScript. After the webpage is loaded, data is constantly being retrieved from the server. Javascipt allows this data to dynamically change the webpage, all without refreshing the page. The charts are rendered via Chart.js

Reporting

In the near future, SQL databases will be incorporated to permenately store all data. As of now, the Flask server temporarily stores all data in memory.
