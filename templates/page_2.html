Introduction

Here I share insights into my professional journey within industrial automation, with a particular focus on Programmable Logic Controllers (PLCs), Human-Machine Interfaces (HMIs), and Supervisory Control and Data Acquisition (SCADA) systems.

My journey began at Alfred State, where I pursued and successfully completed an Associate's degree in Electrical Construction and Maintenance. This foundational education provided me with a comprehensive understanding of electrical engineering principles and methodologies, setting the stage for a career filled with innovation.

Transitioning from traditional relay logic systems to modern Programmable Logic Controllers (PLCs) was a turning point for my career. PLCs provided unparalleled flexibility and efficiency, simplifying control logic design and enabling rapid adjustments without extensive rewiring. Their modular design and user-friendly interfaces facilitated iterative development and real-time adjustments, in turn allowing innovation and adaptability.

PLCs facilitate seamless integration with robust industrial components, enabling efficient communication and coordination between different systems. These industrial computers hold and execute the program that controls the field devices(motors, pumps, solenoids) by interfacing directly with the field devices or indirectly through RIO(remote controllers typically held near field devices), VFDs(Variable Frequency Drives),Servo Drives,soft starts, and any other inputs/outputs conventional relay logic would allow.

Overview

As part of my project portfolio, I developed a real-time monitoring and control interface using Flaskâ€”a lightweight web framework in Python. This application provides a user-friendly interface for monitoring and controlling vacuum pump systems in industrial settings. Leveraging Flask's flexibility and simplicity, I designed a responsive web UI that communicates with a Programmable Logic Controller (PLC) to fetch real-time data and control parameters of multiple vacuum pumps.

Key Features

Real-time Data Retrieval: The application fetches real-time data from the PLC, including vacuum levels and separator pressures for each pump.

Alarm Status Monitoring: Users can monitor the alarm status of each pump, facilitating quick identification of potential issues or anomalies.

Dynamic UI Elements: The UI dynamically updates pump animations and colors based on their operational status, providing intuitive visual feedback to users.

Technologies Used:

  • Python Flask: For building the web application and handling HTTP requests.
  • JavaScript (AJAX): For asynchronous data retrieval and dynamic UI updates without page reloads.
  • Pylogix Library: For interfacing with the PLC and retrieving real-time data.
  • HTML/CSS: For designing the user interface and styling elements for improved usability.

Project Impact:

This project showcases my ability to develop practical solutions for industrial automation challenges by integrating software and hardware components. By creating a robust monitoring and control interface, I demonstrated proficiency in web development, PLC programming, and data communication protocols. The application's user-friendly design and real-time data visualization capabilities enhance operational efficiency and facilitate proactive maintenance, contributing to improved productivity and reliability in industrial environments.
